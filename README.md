# Multimedia Sentiment Analysis Project

This project performs **sentiment analysis on text and images** using
HuggingFace Transformers, WordCloud visualization, and CLIP.

## Features
- Text sentiment analysis (Positive / Negative)
- WordCloud generation from text
- Image sentiment analysis using CLIP
- Visualization with Matplotlib

## Technologies Used
- Python
- Transformers (HuggingFace)
- PyTorch
- WordCloud
- PIL (Pillow)
- Matplotlib
